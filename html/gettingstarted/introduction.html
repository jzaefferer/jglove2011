Introduction

The JGlove software was developed for the X-IST USB GloveBox and its devices (e.g. X-IST DataGlove, X-IST DataFootsole).
The JGlove Software processes the data send by the GloveBox.
JGlove provides a graphical real-time monitoring, an advanced gesture recognition and signal filtering.
Sensors can be calibrated and all incoming and generated data can be stored with the help of the Log output.
An advanced feature of JGlove is the real-time generating of MIDI message from Glovebox sensors, gestures and mouse sensors.

Main features of the JGlove software are:
  • Individual selections of sensors
  • Sensor calibration
  • Graphical real-time monitoring of all sensors and gestures as bar chart and curve diagramm
  • Generating MIDI message of all sensors and gestures
  • Simple and approximated gesture recognition
  • Log output writing
  • Mouse Sensoring
  • Filtering of sensors
