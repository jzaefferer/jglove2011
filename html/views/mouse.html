Mouse

JGlove offers the possibilty to use your standard PC mouse as a sensors. This can be helpful for testing the MIDI functionality when no other input device is available.
Please note that this is no replacement, the JGlove window must remain visible to use the mouse x and y axis and must have focus to use the mouse buttons.

  1. Open the Mouse Pad View by clicking the main toolbar ( ) icon or use the "Open Mouse pad" option
  2. To activate all mouse motions und buttons press the ( ) icon in the toolbar of the Mouse
    Pad view.
    If the mouse pad view is activated, the monitor views shows a "Mouse X" source for the x-axis,
    a "Mouse Y" source for the y-axis and five mouse buttons "Mouse B1".."Mouse B5"
  3. If your move your mouse inside the Mouse Pad view or press the mouse buttons while your mouse
    is within the Mouse Pad view you all motions are recognized.
Image: MousePad with Monitors
