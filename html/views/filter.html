Filter

I. When should I use filters?

JGlove provides different filter types to influence the source values.
The standard filter types ( tilt and smoothing filter) have the purpose of stabilizing the incoming values.

Image: Two tilt sensors with no filter example
Image: Two tilt sensors with tiltfilter example (duration 8)

The other two filter types (modulation and random Filter) can be applied for a more artificial and creative usage. With the help of the modulation filter you can for example realize a LFO (low frequency oscillator). II. Standard filter types

  1. Tilt filter

    The Tilt filter was specially created for tilt sensors. Tilt sensors have the disadvantage that the values are vibrating.
    To minimize that problem JGlove offers a filter that stabilizes the signal through interpolation. The filter causes a small latency, depending on the Duration parameter value.

    Parameters:
    • Duration:
      This parameter influences how much the signal is stabilized. The higher the duration value, the more stabilized is the signal.
      Please also consider: The higher the duration value, the higher is the delay
      Choose as possible a low parameter value to avoid delay problems.

  2. Smoothing filter

    Like the Tilt filter the Smoothing filter stabilizes the signal. The filter reduces trouble signals and fast skips and smoothes the signal with in- and outfadings. In contrast to the Tilt filter this filter works without latency.

    Parameters:
    • Duration:
      This parameter influences how much the signal is stabilized.
      The higher the duration value, the longer are the fadings.
    • Cut:
      This parameter specifies at which value skips the filter starts to works. If the difference of two successiv value is higher than the cut value the filter starts working by fading the signal.

III. Artifical filter types

  1. Modulation filter

    The Modulation filter adds a sinus modulation to the incoming values. With the help of the filter type you can create special effects like e.g. a LF0 (low frequency oscillator) when choosing a Pitch Bend or a VCO (velocity oscillator) when choosing a Volume Control in the MIDI Configuration view (-> MIDI Configuration).

    Parameters:
    • Duration: This parameter influences the wave length of the sinus modulation.
    • Cut: This parameter influences the amplitude of the sinus wave which will be added to the orignal value.

  2. Random filter

    This filter adds and substracts random values.
    This filter has no parameters (subject of change).

IV. How to include filters.

  1. Open the Monitor View or the Sensor Setup View
  2. Click the "spanner" button of any sensor.
    Depending on the kind of sensor you choose, a dialog will open, containing the following elements:
    Image: Filter Viewer
  3. Choose a filter type by pressing the "Add..." button.
    4. A dialog that lets you choose a filter will open:
    Image: Filter Viewer Add Dialog
  4. If you like you change filter parameters by clicking in the parameter field and then clicking on the appearing "..." button. Now you are able to change all available parameters.
    Image: Filter Viewer Options Dialog
  5. If you like you can rename the filter name by clicking into the name field of the filter.
  6. By clicking the checkbox button you are able to deactivate and activate a filter.
  7. A marked filter can de deleted by pressing the "Remove" Button.
  8. You are able to include several filters. If you want to add more filters repeat step 3.
Tip: The use of filter causes processor performance. To keep the processor performance of JGlove low, use filters only if necessary.
